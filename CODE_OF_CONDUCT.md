# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc21] as its Code of Conduct.
Participation in DeviceGuard SDK communities — issue tracker, pull requests, discussions,
and any other space where contributors interact — is governed by its standards.

The full, authoritative text lives at the upstream link below and is the reference version.

[cc21]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Instances of behavior that violate the Contributor Covenant may be reported to the project
maintainer at **hoaidongit5@gmail.com**. All reports will be reviewed and investigated
promptly and fairly. Reporter identity is kept confidential.

Community leaders are obligated to respect the privacy and security of the reporter of any
incident.

## Scope

This Code of Conduct applies within all community spaces, and also applies when an
individual is officially representing the community in public spaces — e.g., using an
official project email address, posting via an official social media account, or acting as
an appointed representative at an online or offline event.
